CSS Statements

CSS Statements are made of Main components

  1. Declaration the Statement as a whole with curly braces.
  2. Selector: the name of the part you would want to style.
  3. Property: properties to edit (e.g. width, height, background-color)
  4. Value: values that we assign to properties (e.g. measurement, color, urls)
